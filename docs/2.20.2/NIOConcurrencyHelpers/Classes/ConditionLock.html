ConditionLock

public final class ConditionLock<T> where T : Equatable

A Lock with a built-in state variable.

This class provides a convenience addition to Lock: it provides the ability to wait until the state variable is set to a specific value to acquire the lock.

  • value

    The value of the state variable.

    Obtaining the value of the state variable requires acquiring the lock. This means that it is not safe to access this property while holding the lock: it is only safe to use it when not holding it.
